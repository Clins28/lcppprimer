# 类的其他特性
##
### 内联成员函数
>定义在类内部的成员函数是自动inline的， 因此，类的构造函数默认是inline函数<br>
>内联函数可以避开函数调用开销，在调用的时候直接内联地展开
```cpp
cout << shorterString(s1, s2) << endl;
cout << (s1.size() < s2.size() ? s1 : s2) << endl;
```
### mutable 可变数据成员
声明为mutable的变量即使在const成员函数中也可以改变值
